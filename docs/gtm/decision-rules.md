# Sirrius — Règles de décision (Start / Pro / Max / NO GO)

## NO GO immédiat (on refuse)
- Signal non exploitable (aucun calendrier / aucun e-mail exploitable).
- Aucun responsable / process de confirmation inexistant.
- Refus des SMS sobres ou refus de STOP.
- Attente "bouton magique" sans accepter un minimum de mise en place.

## Start (Activation 60 jours)
Proposer Start si :
- Il existe des trous <24h (même modestes) OU l'incertitude est forte,
- Et le cabinet accepte une phase d'activation + mesure + preuve sur une fenêtre en jours d'ouverture.

Start = mise en place + preuve, sans objectif de taux.

## Pro (290 CHF / mois / site)
Proposer Pro si :
- Cabinet petit à moyen, besoin simple ("que ça tourne proprement"),
- Jusqu'à 2 praticiens inclus, cap dur max 3 praticiens (add-on +150 CHF, max 1),
- Pas de demande forte de règles avancées / pilotage fin.

Pro = automatisation + réservation unique + preuve horodatée.

## Max (590 CHF / mois / site)
Proposer Max si :
- Besoin de contrôle avancé (répartition équitable, exceptions avancées, plusieurs scénarios),
- Besoin de support prioritaire / tableau de bord complet,
- Ou cabinet plus structuré.
- Jusqu'à 5 praticiens inclus, add-on +120 CHF au-delà de 5.

Max = Pro + contrôle avancé + objectif mesuré (si éligible).

## Rappels pricing (pour éviter les erreurs sur le terrain)
- Start : 240 CHF / mois × 2 (non renouvelable)
- Pro : 290 CHF / mois / site (jusqu'à 2 praticiens, max 3)
- Max : 590 CHF / mois / site (jusqu'à 5 praticiens)
- Site supplémentaire : +300 CHF / mois

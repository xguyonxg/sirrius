# Sirrius — Pricing v1.3 (Start / Pro / Max)

## Vue d'ensemble

| Plan | Prix | Praticiens inclus | Sites |
|------|------|-------------------|-------|
| **Start** | 240 CHF/mois × 2 mois | Jusqu'à 2 | 1 site |
| **Pro** | 290 CHF/mois/site | Jusqu'à 2 (max 3) | Multi-sites |
| **Max** | 590 CHF/mois/site | Jusqu'à 5 | Multi-sites |

## Start (Programme d'activation 60 jours)

- **Prix** : 240 CHF/mois × 2 mois (480 CHF total)
- **Durée** : 60 jours, non renouvelable
- **Scope** : 1 site, jusqu'à 2 praticiens
- **Inclus** : 1 séquence standard, mise en place, preuve horodatée
- **Exclus** : personnalisation avancée, objectif de taux

**Objectif Start** : validation du signal, constitution de la supply, preuve de fonctionnement.

## Pro

- **Prix** : 290 CHF/mois/site
- **Praticiens inclus** : 2
- **Cap dur** : maximum 3 praticiens sur Pro
- **Add-on praticien Pro** : +150 CHF/mois (max 1 add-on)
- **Add-on site** : +300 CHF/mois

**Inclus** :
- Automatisation complète
- Réservation unique
- Preuve horodatée (historique complet)
- Pause entre messages (anti-spam)
- Plafond anti-sollicitation : 1 offre/jour/patient, 3 offres/7 jours/patient
- Fenêtre d'envoi configurable (08:00–20:00 par défaut, option 24/7)
- Confirmation = écriture dans l'agenda du cabinet (sans données sensibles)

## Max

- **Prix** : 590 CHF/mois/site
- **Praticiens inclus** : 5
- **Add-on praticien Max** : +120 CHF/mois (au-delà de 5)
- **Add-on site** : +300 CHF/mois

**Inclus (tout Pro, plus)** :
- Répartition équitable entre praticiens (tri "moins sollicités d'abord")
- Règles avancées : cooldowns post-confirmation, blacklist/pause patients, exceptions fines, 2–3 scénarios messages
- Objectif mesuré (si éligible, voir engagements)
- Support prioritaire

## Pro vs Max en une phrase

**Pro** = automatisation fiable avec preuve.
**Max** = Pro + contrôle avancé + objectif mesuré conditionnel.

## Vocabulaire à utiliser

| Terme correct | Signification |
|---------------|---------------|
| Réservation unique | Un seul patient peut confirmer un créneau. |
| Preuve horodatée | Chaque action est tracée avec horodatage (historique complet). |
| Pause entre messages | Délai anti-spam entre les SMS d'une même vague. |
| Exceptions | Règles d'exclusion (patients, créneaux, horaires). |
| Répartition équitable | Distribution des créneaux récupérés entre praticiens. |

## Notes

- Tous les prix sont en CHF, hors taxes.
- Aucune promesse de ROI monétaire sur Start/Pro.
- Max peut mentionner "objectif conditionnel", jamais "ROI garanti".
